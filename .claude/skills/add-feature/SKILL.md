# 新機能の追加

以下の手順で新機能を実装してください。

## 1. プロジェクト理解
- README.mdを読んで、プロジェクトの全体像を把握する
- architecture.mdを確認して、レイヤー構造を理解する

## 2. 既存パターンの調査
- Grepを使って、類似する既存機能を検索する
- 見つかった実装ファイルを読む
- 以下の点を確認する：
　- サービス層の呼び出し方
　- エラーハンドリングのパターン
　- テストの書き方

## 3. 実装
- 既存パターンに従って、新機能を実装する
- architecture.mdのレイヤー構造を遵守する

## 4. テストと検証

### テストの作成

- テストは既存のテストディレクトリに配置する
- 既存のテストがあれば、そのスタイルに従う

**テスト観点:**
- 正常系（基本パス）
- エッジケース（空入力、境界値）
- エラーケース（不正入力、想定外のデータ）

### ビルドと検証

- CLAUDE.md に記載されたビルドコマンドを実行する
- エラーが出た場合：
　- CLAUDE.md のアーキテクチャセクションを再確認
　- 既存パターンとの差異を確認
　- 修正して再ビルド

## 5. ドキュメント更新
- 必要に応じて、関連ドキュメントを更新する
