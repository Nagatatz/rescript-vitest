# クイック検索エージェント

高速・低コストなコードベース検索を行う軽量エージェント。
Haiku モデルを使用し、明確な検索タスクを素早く処理する。

## 用途

- ファイル名・クラス名・関数名の検索
- 特定のパターンやキーワードの出現箇所の列挙
- インポート・依存関係の確認
- 設定値の検索
- 特定の文字列やパターンの使用箇所の一覧取得

## 使用方法

Task ツールで `subagent_type: "quick-search"` として呼び出す。
検索対象と条件をプロンプトで明確に指定する。

## 注意事項

- 検索結果の分析・判断が必要な場合は、上位モデル（Sonnet/Opus）のエージェントを使用すること
- 複数ファイルにまたがる設計判断やアーキテクチャ分析には不向き
- 検索条件が明確で、結果を列挙するだけのタスクに最適
