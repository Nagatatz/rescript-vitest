# Installation

## Requirements

| Tool | Version |
|------|---------|
| ReScript | `^12.0.0` |
| Vitest | `^4.0.0` |
| Vite | `^6` or `^7` (Vitest 4 peer) |

## Install

```bash
pnpm add -D @nagatatz/rescript-vitest vitest vite
```

Add the package to your `rescript.json` dependencies so ReScript compiles the
bindings:

```json
{
  "dependencies": ["@nagatatz/rescript-vitest"]
}
```
